# 🤖 AI-LOG.md — Cosmic Watch

## Purpose

This document logs how Large Language Models (LLMs) were used **to assist** rather than replace original coding efforts during the development of Cosmic Watch, as required by the hackathon rules.

---

## Usage Log

### Entry 1: Project Documentation

- **Date**: 2026-02-12
- **Tool**: AI Coding Assistant
- **Scope**: Documentation generation
- **What was generated**: Comprehensive DOCS/ folder with 20+ documentation files covering architecture, API docs, database schemas, guides, and wireframes
- **Human Contribution**: Problem statement definition, feature requirements, technology selection decisions, project vision
- **AI Contribution**: Structuring documentation, formatting, ASCII diagrams, consistency across documents

### Entry 2: Architecture Design

- **Date**: TBD
- **Tool**: AI Coding Assistant
- **Scope**: System architecture planning
- **What was generated**: Architecture patterns, data flow diagrams, component hierarchy
- **Human Contribution**: Architectural decisions, technology trade-offs, scalability requirements
- **AI Contribution**: Diagram formatting, pattern documentation, best practice suggestions

---

## Guidelines Followed

1. **AI as Assistant**: AI was used to accelerate documentation and boilerplate generation, NOT to replace understanding of the codebase
2. **Human Decision-Making**: All architectural decisions, technology choices, and design patterns were made by the team
3. **Code Review**: All AI-generated code was reviewed, understood, and modified as needed by team members
4. **Original Logic**: Core business logic (risk scoring algorithm, alert evaluation) was designed by the team with AI helping with implementation patterns

---

## Transparency Statement

We believe in transparent use of AI tools. This log is maintained honestly to show that while AI accelerated our development process, the intellectual design, problem-solving, and decision-making remain human-driven. Our team members can explain and defend every line of code in this project.
